Basic ChatBot

Overview

Basic ChatBot is a simple rule-based chatbot built using Python. The chatbot interacts with users through predefined responses based on the input provided.

It can respond to common greetings and questions such as "hello", "how are you", and "what is your name". The conversation continues until the user enters "bye".

Features

- Console-based chatbot
- Dynamic user input
- Predefined responses
- Continuous conversation using loops
- Exit option using "bye"
- Easy to understand and beginner-friendly

Technologies Used

- Python 3

Concepts Used

- Functions
- While Loop
- If-Elif-Else Statements
- Input and Output
- String Handling

How to Run

1. Clone the repository:

git clone https://github.com/your-username/basic-chatbot.git

2. Open the project folder:

cd basic-chatbot

3. Run the Python file:

python chatbot.py

Sample Output

🤖 ChatBot: Hello! Type 'bye' to exit.

You: hello
🤖 ChatBot: Hi!

You: how are you
🤖 ChatBot: I'm fine, thanks!

You: what is your name
🤖 ChatBot: My name is ChatBot.

You: bye
🤖 ChatBot: Goodbye!
